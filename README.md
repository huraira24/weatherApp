# 🌦️ Weather App

A responsive weather application built using HTML, CSS, and JavaScript.

## 🚀 Live Demo
👉 

## 📌 Features
- 🌍 Search weather by city
- 🌡️ Real-time temperature
- 💧 Humidity display
- 📅 5-day forecast
- 🎥 Dynamic video background

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeather API

## 📖 How to Use
1. Enter a city name
2. Click search
3. View weather details
