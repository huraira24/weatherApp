# 🌦️ Weather App

A modern and responsive weather application built using HTML, CSS, and JavaScript that provides real-time weather information and a 5-day forecast for any city. The application integrates a third-party weather API to fetch accurate data, including temperature, humidity, and weather conditions, and dynamically updates the user interface based on user input.

The project features an interactive and visually engaging design, incorporating a full-screen video background and clean card-based layout to enhance user experience. It also includes error handling for invalid inputs and ensures smooth performance across different devices and screen sizes.

This project demonstrates strong fundamentals in API integration, asynchronous JavaScript (fetch/async-await), DOM manipulation, and responsive UI design.

## 📌 Features
- 🌍 Search weather by city
- 🌡️ Real-time temperature
- 💧 Humidity display
- 📅 5-day forecast
- 🎥 Dynamic video background

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeather API

## 📖 How to Use
1. Enter a city name
2. Click search
3. View weather details
